# Livello di piattaforma

- DBMS (Database Management System) → gestisce l’accesso ai dati.
- BPMS (Business Process Management System) → automatizzazione e supporto ai processi.
- UIS (User Interface System) → suite di servizi di gestione di interfacce di vario tipo.
- Rule Engine → definizione, test ed esecuzione delle regole di business.
